h1. MiniWiki

Small plugin to integrate a wiki in your website.

h2. Requirements
* RedCloth (developed with 3.0.4/4.1.9)

<pre><code>gem install RedCloth</code></pre>
And add the following to your @config/environment.rb@
<pre><code>config.gem 'RedCloth', :lib => 'redcloth', :version => '~> 4.1.9'</code></pre>


h2. Install

# @./script/plugin install git://github.com/tompesman/mini_wiki.git@
# @./script/generate mini_wiki migration@
# @rake db:migrate@
# @./script/generate controller wiki@
# Add @map.mini_wiki@ to your @config.routes.rb@

h3. Optionally

you can add the map.root
<pre><code>map.root :controller => "wiki"</code></pre>


h2. Add authentication and username

It's possible to add some security to the wiki, you can override the @authorized?@ and the @username@ method.
The @username@ method will be used as a author when a wiki page is edited. The @authorized?@ will be checked before editing the wiki.

<pre><code>class WikitestController < MiniWikiController
    
  def authorized?
    false
  end

  def username
    'Username'
  end
end
</code></pre>
